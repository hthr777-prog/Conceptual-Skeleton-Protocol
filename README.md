# Basic Usage — CSP

## For Users
Paste `system-prompt.md` as the **first message** to the model  
(or as a *system prompt* if the platform supports it).

## For Developers
Integrate `system-prompt.md` as the **system instruction** in the API call.

## Advanced Usage
If you want the model to use additional components (K, R, L, V, etc.),  
append `component-library.md` **after** the system prompt.

---

## File Overview

- **`concept-edition.md`** — Full CSP protocol (human-readable)
- **`system-prompt.md`** — Instructions for the model (copy & paste)
- **`component-library.md`** — Optional extended components
